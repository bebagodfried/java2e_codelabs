### useful commands

**How to build:** <br>
On linux<br>
`javac -cp ":lib/mysql-connector-j-9.0.0.jar" App.java`

On windows or macOS<br>
`javac -cp ";lib/mysql-connector-j-9.0.0.jar" App.java`

**How to run:**<br>
On linux<br>
`java -cp ":lib/mysql-connector-j-9.0.0.jar" App`

On windows or macOS<br>
`java -cp ";lib/mysql-connector-j-9.0.0.jar" App`
